# She Moves Africa - Vanilla Website

A static website for She Moves Africa, empowering African women through film, fitness, and community building.

## About

She Moves Africa is a movement that empowers African women through storytelling, wellness programs, and community action. This website showcases our mission, films, fitness programs, community hubs, and impact across Kenya and beyond.

## Features

- **Responsive Design**: Mobile-first approach with clean, modern styling
- **Interactive Carousel**: Showcasing community stories and impact
- **Film Portfolio**: Highlighting our documentary work including "With Grace"
- **Community Sections**: About our hubs, fitness programs, and collaboration opportunities
- **Contact Form**: Easy way for visitors to get in touch
- **Performance Optimized**: Fast loading with WebP images and optimized CSS

## Technology Stack

- **HTML5**: Semantic markup with accessibility in mind
- **CSS3**: Modern styling with flexbox/grid layouts and smooth animations
- **Vanilla JavaScript**: Lightweight interactivity for carousel and navigation
- **WebP Images**: Optimized image formats for faster loading

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Or serve via a local web server for development

## Deployment

This is a static website that can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- WordPress via FTP
- Any static hosting service

## Content Sections

- **Home**: Hero section with mission statement and community stories
- **About**: Our story, founder information, and team details
- **Films**: Documentary portfolio including "With Grace"
- **Fitness**: Wellness programs and community activities
- **Hubs & Impact**: Community centers and measurable impact
- **Collaborate**: Partnership opportunities and ways to get involved
- **Press**: Media coverage and press resources
- **Contact**: Contact form and organizational information
